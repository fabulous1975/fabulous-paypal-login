# Fabulous - PayPal Login (Almanya)

⚠️ **BU PROJE TAMAMEN EĞİTİM AMAÇLIDIR** ⚠️  
Bu sayfa, PayPal Almanya giriş sayfasının **görsel bir kopyasıdır**.  
**GERÇEK BİR PAYPAL GİRİŞİ DEĞİLDİR.**  
Kullanıcı adı veya şifre toplamaz, veri göndermez, kötü amaçlı hiçbir kod içermez.

---

## 📁 İçindekiler

- `login.html` – Ana giriş sayfası (Almanca, temiz HTML+CSS)
- `login.php` – Form işleyici (örnek amaçlı, hiçbir veri kaydetmez)
- `Fabulous - PayPal_Files/` – CSS, JS ve görsel dosyaları

## 🛠️ Kullanım

1. Dosyaları bilgisayarına indir.
2. `login.html`'i tarayıcında aç – hemen görürsün.
3. PHP'yi çalıştırmak istersen, XAMPP/WAMP gibi bir sunucu kullan.
4. `login.php` içinde hiçbir veri kaydedilmez, sadece örnek yönlendirme vardır.
5. Örnek Yönlendirmeyi Orijinal sitesine verdim Sadece **ÖRNEKTİR**
